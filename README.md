# ☕ Sistema de Pedidos - Café Expresso

Projeto desenvolvido em Java com foco em lógica de programação, orientação a objetos e controle de versão com Git.

Funcionalidades

- Cadastro de produtos
- Adição de itens ao pedido
- Cálculo de subtotal por item
- Cálculo do valor total do pedido
- Controle de status do pedido (Aberto, Em preparo, Finalizado)

Conceitos aplicados

- Programação Orientada a Objetos (POO)
- Listas (ArrayList)
- Enum para controle de status
- Encapsulamento
- Versionamento com Git

Estrutura

- Produto
- ItemPedido
- Pedido
- StatusPedido

Objetivo

Praticar conceitos fundamentais de desenvolvimento e simular um sistema simples de pedidos.
